## 📌 Sobre o projeto

Este projeto foi desenvolvido durante o bootcamp **Potência Tech Angular Developer - Powered by iFood**, com foco em práticas de desenvolvimento front-end moderno.

A aplicação simula uma **Pokédex interativa**, permitindo explorar Pokémons com uma interface dinâmica, responsiva e visualmente aprimorada.

O projeto foi evoluído com melhorias visuais e funcionais em relação à versão base do bootcamp.

---

## ✨ Melhorias implementadas

### 🎨 Interface & UX
- 🌙 Dark Mode implementado com variáveis CSS
- 🪟 Modal com efeito Glassmorphism
- ✨ Animações suaves e transições
- 🎯 Layout mais moderno e centralizado

---

### 🔎 Funcionalidades
- 🔍 Sistema de busca de Pokémons
- ❤️ Sistema de favoritos com feedback visual
- ⚡ Interações mais fluidas nos cards

---

### 📱 Responsividade
- 📲 Layout adaptado para mobile
- 🧩 Grid responsivo otimizado
- 📌 Header ajustado para telas menores
